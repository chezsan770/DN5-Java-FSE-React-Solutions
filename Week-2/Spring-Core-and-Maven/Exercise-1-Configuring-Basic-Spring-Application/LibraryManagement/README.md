# Exercise 1: Configuring a Basic Spring Application

This project configures a basic Spring application for a library management scenario.

## What Is Included

- Maven project named `LibraryManagement`
- Spring Core dependency through `spring-context`
- XML application context in `src/main/resources/applicationContext.xml`
- `BookService` bean
- `BookRepository` bean
- Main class that loads the Spring context and calls the service
