# Exercise 9: LibraryManagement

## Instructions

1. Open the `LibraryManagement` project in IntelliJ IDEA.
2. Make sure Maven dependencies are loaded.
3. Run `LibraryManagementApplication.java`.
4. Open the H2 console at:

```text
http://localhost:8080/h2-console
```

5. Use this JDBC URL:

```text
jdbc:h2:mem:testdb
```

6. Test the application after it starts.
